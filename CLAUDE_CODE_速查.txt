以下为贴子原文：
——
这是我推荐你安装的一些skill：
1.Superpowers
算是一个复合skill，打包了 20 多个可组合的 Skill，覆盖软件开发的完整流程。brainstorming、TDD、代码审查、Git 提交，每个环节都有对应的 Skill 来约束 Claude 的行为。
你可以直接在打开claude code后，输入以下内容来安装
```
claude plugin install superpowers@claude-plugins-official
```
---
2.Planning with Files
Claude Code 自带的 Plan Mode 有个让人头疼的问题：规划存在对话上下文里，上下文一压缩就丢了。长任务做到一半，Claude 忘了自己在干嘛。又从头开始。
Planning with Files 把规划、进度和知识都写进 Markdown 文件。Claude 开始干活前先创建计划文件，每完成一步就更新进度，遇到有用的信息就记到知识文件里。文件在磁盘上就不会丢，即使上下文被压缩了也能恢复状态。
```
claude plugin marketplace add OthmanAdi/planning-with-files
claude plugin install planning-with-files
```
---
3.UI UX Pro Max
这是一个优化ai制作前端能力的skill，使用它制作的前端会比直接做的更好看/更风格化/更专业。
```
claude plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill
claude plugin install ui-ux-pro-max@ui-ux-pro-max-skill
```
---
4.Playwright
让你的Claude code能够直接操作（当然，没有这句话本身显得那么流弊）你的浏览器来完成一些工作。简单来说，之后你做前端遇到问题，可以让claude来操作浏览器检查元素啥的来帮你判断和解决问题。
```
claude plugin install playwright@claude-plugins-official
```

5.tavernlike
这是我自己写的skill，详情可见https://discord.com/channels/1134557553011998840/1502649849995333693
是本教程的必须安装的skill。
```
/sillytavern-web
```
